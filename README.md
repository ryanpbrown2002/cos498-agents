# cos498-agents

A voice-driven frontend development application. Talk to it and watch it build a webapp frontend in real-time — no buttons, no forms, just conversation.

## Vision

Speak your design intent and watch multiple AI agents collaboratively generate, iterate, and refine a live frontend right in front of you. Rapid feedback loops let you steer the UI with your voice while agents handle the code.

## Dev Environment

This project uses a [Dev Container](https://containers.dev/) with:
- Python 3.12
- Node.js 22
- Claude Code CLI

To get started, open this repo in VS Code with the Dev Containers extension or use GitHub Codespaces.
